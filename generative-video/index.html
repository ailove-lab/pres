Генерация видео

Как получить 1 000 000 роликов за 10 часов

План

  • Кейсы
  • Проблематика
  • Технологии

Пример

  • Пользователь ищет что-то в веб магазине
  • Открывает фейсбук и видит персональную видео рекламу с теми продуктами, что он только что искал

Необходимо

  • Генерировать качественный видео-контент
  • Делать это очень быстро и недорого

Кейсы

  • Инструменты для создания видео-рекламы
  • Персонофицированные видео-открытки
  • Интерактивные видео-инсталляции
  • Проекты с пользовательским видео
  • Брендированные видео-чаты
  • Интерактивные лайв-стримы

Рекламные инструменты

  • Reklamixer (2009)
  • Рендер фермы After effects / Blender / CEF
  • Mobile ad creator
  • Snapchat ad creator

Персонофицированные видео-открытки

  • Nescafe поздравление
  • Megafon имя на пляже
  • GOT - друзья в главных ролях
  • ... очень много их было

Интерактив

  • LG, Axe, Pepsi, Coca-Cola, Winston, Renault, L'Oreal, Mövenpick
  • Дополненная реальность
  • Брендированные игры
  • Виртуальные примерочные

Пользовательский контент

  • Конкурсы "лучшее видео"
  • Стилизация / генерация нового контента
  • Обработка, извлечение инсайтов

Видео-чаты

  • Проекты наподобие чат-рулет
  • "Смерть в сети"

Лайв-стримы

  • Генеративный видестрим в которых аудитория влияет на развитие видео-ряда
  • Дополнение стрима пользовательским интерактивом

Челендж

  • Ресурсоемкость
  • Медленность
  • Tрафик
  • Кроссплатформенность

Ресурсоемкость

  • Создание и сжатие видео очень ресурсоемкий процесс
  • Требует высоких вычислительных мощностей

Медленность

  • Достигнуть реалтайм-генерации очень тяжело достичь
  • Передача контента по сети также требует времени

Трафик

  • Нужно сразу учитывать объёмы аудитории в пике
  • Подбирать CDN исходя из геолокации
  • Подбирать уровень сжатия видео

Платформы

  • Фрагментация устройств и видео-кодеков
  • Необходимость хранить видео под pc/ios/android

Технологии

  • ffmpeg
  • blender
  • after effects
  • cef
  • cairo, skia, opengl

ffmpeg / pros

  • швейцарский нож для работы с видео на низком уровне
  • отлично решает задачи конвертации форматов
  • позволяет решать задачи простого компоузинга

ffmpeg / cons

  • Не подходит для генеративных задач сложнее слайдшоу
  • Документация сложна в освоении

blender / pros

  • полносценный 3D пакет, с хорошим GPU ускорением
  • содержит инструменты для нелинейного видео монтажа
  • полностью автоматизируется на питоне

blender / cons

  • тяжело найти аниматоров

adobe after effects / pros

  • Один из основных инструментов видео-продакшена
  • Отличное качество рендера
  • Огромное ко-во стоковых темплейтов
  • Легко найти аниматора

adobe after effects / cons

  • Тяжелее подается автоматизации
  • Скорость сильо зависит от "красоты" темплейта
  • Windows

cef / pros

  • позволяет создавать видео из обычной web-анимации
  • достаточно быстрo работает, близко к реалтайму

cef / cons

  • качество уступает blender / ae

cairo, skia, opengl

  • Низкоуровневые библиотеки для работы с графикой
  • Позволяют генеррировать видео контент на очень высокой скорости

Структура

Файловaя структура

// Файлы вложены  
index.pug
  settings.pug 
  presentation.pug
    intro.pug
    section_01.pug
    ...
    section_##.pug
    epilogue.pug

index.pug

В этом файле собран базовый шаблон reveal.js

settings.pug

В этом файле собраны повторяющиеся элементы

presentation.pug

В этом файле собраны секции используемые в презентации

Структура простой секции

// section_##.pug
section
  h2 Заголовок секции
  p Содержание секции
  ul
    li список #1
    li список #2
    li список #3

Структура сложной секции

// section_##.pug
section
  section
    h2 Заголовок секции
  section
    h3 Подзаголовок страницы секции
    p Содержание страницы секции
