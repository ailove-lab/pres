ffmpeg

> ffmpeg -i input.avi output.mp4

ffmpeg / pros

  • швейцарский нож для работы с видео на низком уровне
  • отлично решает задачи конвертации форматов
  • позволяет решать задачи простого компоузинга

ffmpeg / cons

  • Не подходит для генеративных задач сложнее слайдшоу
  • Документация сложна в освоении

blender

blender / pros

  • открытый и бесплатный
  • полносценный 3D пакет, с хорошим GPU ускорением
  • содержит инструменты для нелинейного видео монтажа
  • полностью автоматизируется на питоне

blender / cons

  • тяжело найти аниматоров

adobe after effects

adobe after effects / pros

  • Один из основных инструментов видео-продакшена
  • Отличное качество рендера
  • Огромное ко-во стоковых темплейтов
  • Легко найти аниматора

adobe after effects / cons

  • платный
  • Тяжелее подается автоматизации
  • Скорость сильо зависит от "красоты" темплейта
  • Windows

cef

cef / pros

  • позволяет создавать видео из обычной web-анимации
  • достаточно быстрo работает, близко к реалтайму

cef / cons

  • качество уступает blender / ae

cairo, skia, opengl

  • Низкоуровневые библиотеки для работы с графикой
  • Позволяют генеррировать видео контент на очень высокой скорости
