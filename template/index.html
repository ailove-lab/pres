Введение

Данный темплейт собран на базе проекта reveal.js

reveal.js - библиотека, пoзволяющая собирать интерактивные презентации

В данном темплейте активно используется язык html темплейтов pug

Использование pug позволяет разбить презентацию на логические блоки-файлы.

Значительно упростить верстку презентации.

Пример pug верстки

{intro.pug}
section
  section
    h2 Введение
    +arr
  section
    +p Данный темплейт собран на базе проекта reveal.js
    ...
    +p Значительно упростить верстку презентации.
  section
    h3 Пример pug верстки
    pre.
      ...

Запуск

Сборка презентации осущесвляется парсером разметки pug 

> npm install -g pug
> pug -w index.pug

Для тестирования подойдет любой веб сервер для статики

Настройки

Общие переменные могут быть настроены в файле settings.pug

Миксины

В файле mixins.pug заданные некоторые миксины упрощающие оформление:

  • +li - появляющийся li элемент
  • +p - появляющийся параграф
  • +py - питоноский код
  • +arr - стрелка вниз
  • +note - невидимая заметка

Структура

Файловaя структура

// Файлы вложены  
index.pug
  settings.pug 
  presentation.pug
    intro.pug
    section_01.pug
    ...
    section_##.pug
    epilogue.pug

index.pug

В этом файле собран базовый шаблон reveal.js

settings.pug

В этом файле собраны повторяющиеся элементы

presentation.pug

В этом файле собраны секции используемые в презентации

Структура простой секции

// section_##.pug
section
  h2 Заголовок секции
  p Содержание секции
  ul
    li список #1
    li список #2
    li список #3

Структура сложной секции

// section_##.pug
section
  section
    h2 Заголовок секции
  section
    h3 Подзаголовок страницы секции
    p Содержание страницы секции
